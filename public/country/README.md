# Banderas (country)

Colocá aquí las banderas con estos nombres (código ISO del país):

- **ar.png** – Argentina  
- **bo.png** – Bolivia  
- **br.png** – Brasil  
- **cl.png** – Chile  
- **co.png** – Colombia  
- **ec.png** – Ecuador  
- **pe.png** – Perú  
- **py.png** – Paraguay  
- **uy.png** – Uruguay  
- **ve.png** – Venezuela  

Recomendado: 40px de ancho aprox. (ej. 40×28 o similar).
